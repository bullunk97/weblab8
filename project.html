Panel Otomatisasi Penerangan

Kelompok 4 - Semester 4 Juni 2024 Otomasi, Penerangan, Sensor

Project ini merupakan penerapan dari modul instalasi penerangan dengan penggunaan sensor dan sistem otomatis. Panel ini dirancang untuk menyalakan lampu secara otomatis berdasarkan intensitas cahaya dan waktu.

Mahasiswa belajar merancang rangkaian, mengukur daya, serta memprogram kontroler sederhana. Project ini mengintegrasikan berbagai komponen yang telah dipelajari di laboratorium.

Komponen Utama
  • Sensor LDR (Light Dependent Resistor)
  • Timer Relay Digital
  • Kontaktor 220V
  • MCB 1P 10A
  • Lampu LED 20W
  • Push Button Start/Stop
  • PLC Mikro (Optional)
  • Power Supply 12VDC
Capaian Pembelajaran
  • Pemahaman sistem kontrol otomatis
  • Kemampuan merancang diagram pengawatan
  • Pemrograman dasar timer dan sensor
  • Pengukuran dan analisis daya listrik

Sistem Starting Star-Delta Motor 3 Fasa

Kelompok 2 - Semester 5 September 2024 Motor Listrik, Kontrol, 3 Fasa

Implementasi sistem starting star-delta untuk motor induksi 3 fasa 1.5kW. Project ini mensimulasikan pengasutan motor besar untuk mengurangi arus starting yang tinggi.

Mahasiswa mempelajari prinsip kerja, wiring diagram, dan troubleshooting sistem star-delta yang banyak digunakan di industri.

Komponen Utama
  • Motor Induksi 3 Fasa 1.5kW
  • 3 Kontaktor (Main, Star, Delta)
  • Timer Omron H3CR
  • Thermal Overload Relay
  • Push Button dan Pilot Lamp

Prototipe Sistem Smart Home

Kelompok 7 - Semester 6 Desember 2024 IoT, Otomasi, Smart Home

Prototipe sistem rumah pintar dengan kontrol melalui smartphone dan sensor. Sistem ini mengintegrasikan kontrol penerangan, monitoring daya, dan keamanan.

Project ini menggabungkan pengetahuan instalasi listrik dengan teknologi IoT (Internet of Things) terkini.

Fitur Utama
  • Kontrol lampu via smartphone
  • Monitoring konsumsi daya real-time
  • Sensor gerak untuk keamanan
  • Sistem notifikasi via Telegram
Teknologi Digunakan
  • NodeMCU ESP8266
  • Sensor PZEM-004T
  • Relay Modul 4 Channel
  • Platform Blynk/Home Assistant
